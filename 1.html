    ここで、銀河素数の条件である
    “基数Nにおいて最小素数と最大素数の和が全ての桁でNになる”
    を満たすためには、
    最大ギンガ素数側のC部分は0に足してNになる数、つまりC=Nとなる必要があるが、
    1桁のN進数の最大値はN-1なので、条件を満たす最大ギンガ素数は存在しない。

    つまり、銀河素数が存在する為には、
    最小最大ギンガ数のいずれの桁にも0を含まない事が必要条件だが、
    基数が大きくなると最小ギンガ素数側のC部分に0を含む傾向が高くなるので、
    銀河素数が存在しない可能性が高い、と言える。

    （正確な値とは厳密には異なるが、ほぼ以下の通り。）
    基数=nの場合、最小側のCが0になる確率＝ \( 1-{ {\left( 1-\frac{ 1 } { \ln{ ({n}^{11}) } } \right)}^{n-3} } \)

    ABABCCBCBADBは基数が大きくなる程、最小素数側のCが0になる確率が上がる。
    なので、素数定理と極端に乖離した、異常な間隔の素数分布が発生しない限り、
    銀河素数の存在は期待出来ない。
