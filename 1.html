基数201以上の場合
基数4～200の計算結果について、最小ギンガ素数に着目してみると、
基数50前後を境に、C部分の数字が殆ど0になっている事が解ります。
探しているギンガ素数の特徴は“最小素数と最大素数の和が全ての桁でNになる”なので、
最小ギンガ素数側のC部分が0なら、最大ギンガ素数側のC部分は0に足してNになる数、つまりC=Nとなる必要がありますが、
1桁のN進数の最大値はN-1なので、条件を満たす最大ギンガ素数は存在しない事になります。

つまり、銀河素数が存在する為には、
最小最大ギンガ数のいずれの桁にも0を含まない事が必要条件になりますが、
基数が大きくなると最小ギンガ素数側のC部分に0を含む傾向が高くなるので、
銀河素数が存在しない可能性が高い、と言えます。

最小ギンガ素数側のC部分に0を含む確率

最小ギンガ素数側のC部分に0を含む確率は、
素数定理を用いて大ざっぱに計算すると以下の通りになります。
（正確に計算すると異なりますが、ほぼ以下の通りです。）
基数=nの場合、最小側のCが0になる確率＝ \( 1-{ {\left( 1-\frac{ 1 } { \ln{ ({n}^{11}) } } \right)}^{n-3} } \) 基数が大きくなる程、最小ギンガ素数側のC部分が0になる確率は100%に近づく為、
素数定理と極端に乖離した、異常な間隔の素数分布が発生しない限り、
最小ギンガ素数と最大ギンガ素数の和が全ての桁でNになる基数は存在しません。
